Features

Everything a coding agent needs — locally

60+ built-in tools, real context management, and an interruptible agent loop. No telemetry, ever.

🔒

Local & private

Every token is generated by your own models on your hardware. No data ever leaves the machine.

💸

Free forever

No subscription, no per-token billing. Run any model your GPU (or CPU) can handle.

📦

Zero dependencies

One vanilla-PHP file built by concatenation. No Composer, no npm, no framework — auditable in one read.

🛡️

Safe by default

Edits show a colorized diff and ask first. Every change is checkpointed and reversible with /undo.

👥

Crew — multi-agent

A Director plans, parallel Coders build in isolated git worktrees, an Auditor reviews. Per-role models, multi-host.

🧩

Skills

Reusable SKILL.md packs with progressive disclosure. Install, export, and share them.

🕸️

Graph memory

A wiki-linked knowledge base the agent recalls on demand — context that survives across sessions.

🔌

MCP & LSP

Connect Model Context Protocol servers (stdio + HTTP/SSE) and get real code navigation & diagnostics.

🖥️

Two local backends

Ollama by default; opt into LM Studio for a higher model ceiling. Both stay on localhost.

🌐

CLI · desktop · web

Run it in the terminal, as a native desktop app, or in your browser (composer serve) — same UI, same local engine, shared data. Use it from your phone via a tunnel.

Install

Up and running in two commands

Requires PHP 8.0+ with curl, and Ollama running locally.

1 Start Ollama

# install a model
ollama pull qwen2.5-coder
ollama serve

2 Build & install

git clone https://github.com/\
  kennethyork/OllamaDev
cd OllamaDev
make install

3 Code

# interactive chat
ollamadev

# or run a crew
ollamadev crew "add tests"

Prefer a GUI? There's a desktop app (ADE), a browser/web mode (run it on a server and use it anywhere), and a VS Code extension too — see CLI · Desktop · Web and the self-hosting docs.
